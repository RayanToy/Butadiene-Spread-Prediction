# Butadiene-Spread-Prediction
Анализ и прогнозирование арбитражных окон на рынке бутадиена
О проекте
Этот проект представляет собой количественное исследование глобального рынка бутадиена с целью выявления арбитражных возможностей между европейским и азиатским регионами. Работа включает в себя статистический анализ исторических данных за период 2011–2023 гг., исследование сезонности и построение моделей временных рядов для прогнозирования ценового спреда.

Основная цель работы — разработать алгоритм оценки рыночной конъюнктуры, способный предсказывать направление и величину разницы цен (спреда) для принятия решений об экспортно-импортных операциях.

Структура проекта
Analysis_Spread.ipynb — основной Jupyter Notebook, содержащий полный цикл исследования: от загрузки и очистки данных до построения прогнозных моделей и оценки их качества.
requirements.txt — список зависимостей и библиотек, необходимых для воспроизведения среды.
README.md — файл с описанием методологии и результатов проекта.
Этапы работы
1. Исследовательский анализ данных (EDA)
Проанализирована динамика цен в Европе и Азии. Рассчитан исторический спред и его относительные показатели.
Выявлена структурная асимметрия рынка: в 85% наблюдений азиатский рынок торгуется с премией к европейскому.
Обнаружен структурный сдвиг (structural break) в рядах данных, вызванный энергетическим кризисом 2021–2022 годов.
2. Статистический анализ и проверка гипотез
Проведена проверка временных рядов на стационарность с использованием тестов Дики-Фуллера (ADF) и KPSS.
Выполнено дифференцирование ряда для устранения трендов.
Анализ автокорреляционных функций (ACF/PACF) подтвердил наличие сезонных паттернов и инерционности рынка.
3. Моделирование и прогнозирование
Были разработаны и сравнены два подхода к прогнозированию:

SARIMAX: Модель сезонной авторегрессии с использованием экзогенной переменной (динамика азиатских цен как опережающий индикатор для Европы).
AutoReg (AR): Модель для выявления чистых циклических закономерностей. В ходе экспериментов установлено, что использование 15-месячного лага (цикл запасов) дает более высокую точность, чем стандартный 12-месячный календарный цикл.
4. Оценка качества и валидация
Модели протестированы на отложенной выборке.
Проведен стресс-тест модели SARIMAX на периоде высокой волатильности (2021–2023 гг.) для оценки устойчивости алгоритма к внешним шокам.
Ключевые результаты
В ходе исследования выявлены следующие закономерности:

Сезонность: Наименьшая доходность экспорта наблюдается в феврале (дно рынка), наибольшая — в ноябре.
Производственные циклы: Рынок подвержен 15-месячному циклу накопления и распродажи запасов (Inventory Cycle), что подтверждается высокой точностью модели AutoReg с соответствующим параметром.
Точность прогнозирования:
Модель AutoReg на стабильном рынке показала R² ≈ 93.3% и среднюю абсолютную ошибку (MAE) 71 $/т.
Модель SARIMAX в условиях кризисного рынка показала R² ≈ 58.1%, что является приемлемым результатом для определения направления тренда в условиях макроэкономической нестабильности.
Как запустить проект
Клонируйте репозиторий:
Bash

git clone <URL-вашего-репозитория>
Установите необходимые библиотеки:
Bash

pip install -r requirements.txt
Запустите Jupyter Notebook:
Bash

jupyter notebook Analysis_Spread.ipynb
Используемые технологии
Python 3.8+
Pandas, NumPy — обработка и подготовка данных.
Statsmodels — статистические тесты, модели SARIMAX и AutoReg.
Scikit-learn — расчет метрик качества (RMSE, MAE, R²).
Matplotlib, Seaborn — визуализация результатов.
